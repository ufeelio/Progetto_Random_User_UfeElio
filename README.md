# Progetto Random User

## Introduzione

Questo progetto è una semplice applicazione web che permette di generare degli utenti in modo casuale attraverso l'utilizzo di un **API** gratuita fornita da <https://randomuser.me>.

## Strumenti Utilizzati

- **HTML**
- **CSS**
- **JS**
- **JQUERY**

## Funzionamento dell'applicazione

L'applicazione all'avvio si presenta i pulsanti ***Genera*** e ***Ordinamenti***, una ***Search Bar***, la ***Griglia Visualizzazione*** degli utenti e i ***Pulsanti di Navigazione*** della griglia.
